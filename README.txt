Este es un repositorio de pruebas.
Ejercicio de Open ClassRoom del curso Gestiona tu código con GIT y GitHub

Este ejercicio te ayudará a poner en práctica algunas de las ideas que aprendiste durante esta sección del curso.

Deberías crear un repositorio simple te Git que contenga:

Un archivo llamado "README.txt" que presente el proyecto.
Al menos otros 2 archivos a tu elección (Pueden contener scripts, texto sobre un proyecto personal, etc.);
Al menos 4 commits separados que muestren cómo añadir y modificar distintos archivos.
